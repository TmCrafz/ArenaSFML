# Controls

## Keyboard
### General:
| Key | Action |
| --- | --- | 
| ESC | Pause Game |
| F2 | Show FPS |
| F3 | Show collision shapes |
| W | Up |
| A | Left |
| S | Down |
| D | Right |
| Right Mouse |	Block |
### Knight
| Key | Action |
| --- | --- | 
| Left Mouse | Attack |
| Left Control + Left Mouse | Strong Attack |
